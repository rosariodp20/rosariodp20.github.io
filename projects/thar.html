Treasure Hunt AR

Enterprise Mobile Application Development project

Treasure Hunt AR was developed as a project for the Enterprise Mobile Application Development (EMAD) course. The EMAD course provides fundamental methodological and technological tools for studying enterprise applications and cross-platform technologies. It also includes a group project, with each team being assigned a corporate coach from national and international companies such as Accenture, Google, IBM, I.T.Svil, Reply, Nexsoft, NTT Data, and others. For the development of Treasure Hunt AR, my team and I collaborated with Nexsoft, aiming to create an augmented reality treasure hunt.

Application home screen

The application featured two types of users: the treasure hunt organizer and the participant. The organizer could create a treasure hunt, set the number of clues, and provide descriptions for them. During this process, the system generated a unique ID and password for the hunt, allowing the organizer to modify clues if necessary. Additionally, the system produced a PDF file containing all the targets to be scanned during the treasure hunt.

To participate, users simply had to enter the treasure hunt code provided by the organizer and start scanning targets found during the game.

Example of scanning a target

The application was developed using Flutter with Dart for both backend and frontend management, while Vuforia, ARCore, and Unity were used for handling the augmented reality components.
