Carta Natura

Geographical Information System project

Carta Natura was developed as a project for the Geographical Information System course. The system was created in collaboration with the Department of Civil Engineering at UNISA to automate a research process previously conducted manually by the department.

The product is a web application designed to analyse natural ecosystems in the Campania region. Users can select specific areas of land or municipalities within the provinces of Salerno, Benevento, Caserta, Avellino, and Naples, then proceed with an environmental analysis to estimate the monetary value of CO2 absorption in the selected area.

A screen of Carta Natura

The Department of Civil Engineering provided CO2 absorption estimates for different plant species per hectometer squared. These data were then processed to enable various analyses using the Carta Natura software.

The system was developed entirely in JavaScript with HTML and CSS for the web application, while Python was used for geo-data analysis and processing, primarily leveraging the GeoPandas, Leaflet and OpenStreetMaps library. Later, the project was integrated with Django and encapsulated in a Docker container to facilitate hosting on the LabGIS UNISA website.
